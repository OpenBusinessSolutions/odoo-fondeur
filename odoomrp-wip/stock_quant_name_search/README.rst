==========================
Quant extended name search
==========================

This module creates a working "name_search" for stock.quant object.

By default stock.quant has a computed name so it is not searchable, the
function uses product code, UoM and lot name. So with this module when
searching by name one quant, it will do a name search in those three objects,
and show the corresponding quants.
