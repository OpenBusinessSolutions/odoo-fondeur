Stock Easy Internal Transfers

The purpose of this module is to simplify internal transfers , for it was created a new form view with the minimum fields required to process an internal transfer.

In addition, was added a inventory validation based in source location, and a new section 'quick internal transfers' it was added in All operations. To enable this feature you must check 'user quick internal transfer in this warehouse' in warehouse form.
